# 测试用例（当前已实现）

## 1. BOOT-001 回归
- 用例：`GET /api/ping` 返回统一响应结构 + `X-Trace-Id`
- 用例：`POST /api/ping/echo` 空参数返回统一校验错误
- 用例：`GET /actuator/health` 可访问且返回 `UP`
- 用例：`GET /swagger-ui/index.html` 可访问

对应自动化测试：`Boot001IntegrationTests`

## 2. AUTH-001 认证与权限
- 用例：`teacher/teacher123` 登录成功，返回 JWT
- 用例：携带 JWT 调用 `/api/auth/me` 返回当前用户和角色
- 用例：学生调用教师创建任务接口返回 403

对应自动化测试：`AuthAndWorkflowIntegrationTests.loginAndMeShouldWork`、`studentShouldNotCreateTask`

## 3. M2 报告主链后端闭环
- 用例：教师创建任务成功
- 用例：学生提交报告成功，版本号从 1 开始
- 用例：教师对提交进行评分和评语
- 用例：学生可查看自己的评语
- 用例：教师导出成绩 CSV 成功

对应自动化测试：`AuthAndWorkflowIntegrationTests.workflowShouldRunThroughTaskSubmissionReviewAndExport`

## 4. 执行方式
在 `lab-flow-report-system` 目录运行：

```powershell
.\mvnw.cmd -U clean test
```

预期：
- `Tests run: 8, Failures: 0, Errors: 0, Skipped: 0`
- `BUILD SUCCESS`
