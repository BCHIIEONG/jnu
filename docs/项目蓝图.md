# 项目蓝图（1页即可）

## 1. 项目一句话
基于 **Spring Boot + Vue3 + MySQL** 的实验课实验流程与报告管理系统（PC + 手机 H5），实现“任务发布—签到—过程记录—报告提交—批阅反馈—统计导出”闭环。

## 2. 角色与边界
- 学生：签到、过程记录、报告编辑/提交、查看反馈
- 教师：发布任务、开启签到/补签、批阅评分、导出
- 管理员：用户/角色权限、基础数据、设备台账、日志/备份

## 3. MVP 优先闭环（建议）
1) 登录/RBAC
2) 任务发布 → 报告提交（Markdown+附件+版本） → 教师批阅评分 → 学生查看反馈 → 成绩导出
3) 二维码签到（短时 token，重复校验）

## 4. 模块地图
认证授权｜课程/任务｜签到｜报告（版本/附件）｜批阅评分｜导出｜设备｜进度记录｜审计日志
