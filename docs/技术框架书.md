# 技术框架书（模板）

## 1. 技术栈
- 前端：Vue3（PC + H5 响应式）
- 后端：Spring Boot（RESTful API）
- 数据库：MySQL
- 文件存储：本地存储（可扩展对象存储）
- 鉴权：JWT + RBAC（学生/教师/管理员）

## 2. 总体架构
- 前后端分离
- 模块划分：认证授权、课程/任务、签到、报告、设备、进度、批阅、导出、审计日志

## 3. 关键数据流
- 登录→获取 token→访问受保护资源
- 教师发布任务→学生提交报告→教师批阅→成绩导出
- 教师开启签到→生成动态 token→学生扫码签到→名单实时查看→补签/缺勤

## 4. 非功能设计
- 安全：BCrypt、权限校验、上传限制、审计日志
- 可部署：可重复的部署步骤（脚本化/容器化按条件）
- 可测试：关键路径用例
