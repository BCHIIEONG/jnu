# Codex 任务单（Append-only）

> 每个任务都要写清：范围、接口、表结构、验收标准（DoD）。

## 任务格式
- Task-ID：例如 BOOT-001
- 目标：一句话
- 范围：包含/不包含
- 依赖：依赖哪些任务/哪些文档
- 交付物：代码/文档/脚本
- 接口：列 endpoint 或 OpenAPI 片段
- 数据表：表名 + 关键字段（或引用 docs/接口与数据库.md）
- 验收标准（DoD）：Given/When/Then（必须可验证）
